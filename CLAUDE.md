# Claude Instructions

## Project: n8n-assistant

This project manages n8n workflows via Terraform for home lab automation.

## Pull Request Workflow

When instructed to push changes, always create a pull request instead of pushing directly:

1. Create a new branch for the changes
2. Commit the changes
3. Push the branch to remote
4. Create a pull request using `gh pr create`

Do not push directly to `main`.
